# To-do List webapp 
This project is a practice project to learn how to use GraphQL in Nuxt 3 and Nestjs.

# Run and Test
### Commands
You can run application using docker compose commands:
```shell
# To run application
docker compose up -d

# To Stop application
docker compose down

# To run application tests
### COMING SOON

# To build and deploy application
### COMING SOON
```
